Gestión de citas – Clinic Dental

Aplicación web desarrollada en JavaScript para la gestión de citas de una clínica dental.
Permite crear, listar, editar y eliminar citas sin utilizar base de datos, haciendo uso de
cookies y LocalStorage para la persistencia de la información.

Proyecto realizado como trabajo de enfoque del módulo  
Ciclo Formativo de Grado Superior DAW.



Funcionalidades

- Alta de nuevas citas mediante formulario.
- Listado de citas en una tabla dinámica.
- Edición de citas existentes.
- Eliminación de citas con confirmación.
- Validación de datos en el lado cliente.
- Persistencia de datos mediante cookies.
- Uso de LocalStorage para la edición de citas por identificador.
- Interfaz responsive y clara.


Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (ES6)
- Cookies del navegador
- LocalStorage


 Estructura del proyecto
index.html -> Listado de citas
cita.html -> Formulario de alta y edición
app.js -> Lógica de la aplicación
styles.css -> Estilos de la interfaz


Uso de la aplicación

1. Abrir index.html en un navegador web.
2. Pulsar en Crear cita para añadir una nueva.
3. Rellenar el formulario y guardar.
4. Desde el listado se pueden editar o eliminar las citas creadas.
